【题解】P2571 [SCOI2010]传送带

 2021-06-28 2021-06-28 基础算法 / 题解 / 省选 / 三分 / 练习题 / SCOI / 一 5 分钟读完 (大约684个字)

Des

在一个 \(2\) 维平面上有两条传送带，每一条传送带可以看成是一条线段。两条传送带分别为线段 \(\text{AB}\) 和线段 \(\text{CD}\)。lxhgww 在 \(\text{AB}\) 上的移动速度为 \(P\)，在 \(\text{CD}\) 上的移动速度为 \(Q\)，在平面上的移动速度 \(R\)。现在 lxhgww 想从 \(\text A\) 点走到 \(\text D\) 点，他想知道最少需要走多长时间。

\(\texttt{Data Range:}\)

对于 \(100\%\) 的数据，\(1\le A_x,A_y,B_x,B_y,C_x,C_y,D_x,D_y\le10^3\)，\(1\le P,Q,R\le10\)。

Sol

设在 \(AB\) 上走到 \(M\) 点离开 \(AB\)，在 \(CD\) 上点 \(N\) 开始在 \(CD\) 上行走。

那么显然 \(dis(A,M)/P,dis(M,N)/R,dis(N,D)/Q\) 都是单调函数。而 2 个单调函数相加后会得到单调函数或单峰函数。也就是说当一个点固定，可以用三分求出另一个点使得答案最小的取值。

设 d(x) 表示 \(M\) 在 \(x\) 位置时 \(dis(M,N)/R + d(N,D)/Q\) 的最小值，那么随着 \(x\) 的增加，\(d(x)\) 会减小。

\(d(x)\) 增加，\(dis(A,M)/P\) 减小，加起来仍然是一个单峰函数。也就是说“外层”仍然可以用三分求解。

这里写的三分和常规方法很不一样。首先在整个值域上取到三等分点，然后检视哪个三等分点的答案最小。找到那个点之后，将值域改为该点左右 eps 的临域。然后把 eps 除以 3。

这样“三分”的正确性可以通过枚举极值点在三分的哪个区域证明。

复杂度 \(O((6\log_3n)^2)\)。

【题解】P1291 [SHOI2002]百事世界杯之旅

 2021-06-24 2021-06-28 数学 / 概率期望 / 题解 / 省选 / SHOI 3 分钟读完 (大约502个字)

Des

“……在 2002 年 6 月之前购买的百事任何饮料的瓶盖上都会有一个百事球星的名字。只要凑齐所有百事球星的名字，就可参加百事世界杯之旅的抽奖活动，获得球星背包，随声听，更可赴日韩观看世界杯。还不赶快行动！你关上电视，心想：假设有 \(n\) 个不同的球星名字，每个名字出现的概率相同，平均需要买几瓶饮料才能凑齐所有的名字呢？

\(\texttt{Data Range:}\)

对于全部的测试点，保证 \(2 \leq n \leq 33\)。

Sol

把总期望拆成得到 \(k\) 转移到 \(k+1\) 个名字的期望之和。设已经得到了 \(k\) 个球星名字，那么得到第 \(k+1\) 个球星名字需要买饮料数的期望是 \((1-\dfrac kn)\times 1+(\dfrac kn)(1-\dfrac kn)\times 2+(\dfrac kn)(1-\dfrac kn)^2 \times 3+\dots\)，然后我就在这卡住了。。。

设买到 \(k\) 个的期望购买次数是 \(f(k)\)，有 \[ f(k)=\frac{n-k+1}n (f(k-1)+1)+\frac{k-1}n \times (?) \] 前半部分很好理解，现在来思考 \(?\) 里填什么。

设 \(f(k)=\sum p_ix_i\)，那么在买到一次已经有的球星后，下一次所有的 \(x_i\) 都会增加 1，所以问号里应该填 \(\sum p_i(x_i+1)\)。

那么最终得到 \[ \begin{align} f(k)&=\frac{n-k+1}n (f(k-1)+1)+\frac{k-1}n \times (f(k)+1)\\ &=f(k-1)+\frac n{n-k+1}. \end{align} \]

【题解】P1850 [NOIP2016 提高组] 换教室

 2021-06-24 2021-06-24 数学 / 概率期望 / 题解 / NOIP / 2016 9 分钟读完 (大约1363个字)

Des

对于刚上大学的牛牛来说，他面临的第一个问题是如何根据实际情况申请合适的课程。

在可以选择的课程中，有 \(2n\) 节课程安排在 \(n\) 个时间段上。在第 \(i\)（\(1 \leq i \leq n\)）个时间段上，两节内容相同的课程同时在不同的地点进行，其中，牛牛预先被安排在教室 \(c_i\) 上课，而另一节课程在教室 \(d_i\) 进行。

在不提交任何申请的情况下，学生们需要按时间段的顺序依次完成所有的 \(n\) 节安排好的课程。如果学生想更换第 \(i\) 节课程的教室，则需要提出申请。若申请通过，学生就可以在第 \(i\) 个时间段去教室 \(d_i\) 上课，否则仍然在教室 \(c_i\) 上课。

由于更换教室的需求太多，申请不一定能获得通过。通过计算，牛牛发现申请更换第 \(i\) 节课程的教室时，申请被通过的概率是一个已知的实数 \(k_i\)，并且对于不同课程的申请，被通过的概率是互相独立的。

学校规定，所有的申请只能在学期开始前一次性提交，并且每个人只能选择至多 \(m\) 节课程进行申请。这意味着牛牛必须一次性决定是否申请更换每节课的教室，而不能根据某些课程的申请结果来决定其他课程是否申请；牛牛可以申请自己最希望更换教室的 \(m\) 门课程，也可以不用完这 \(m\) 个申请的机会，甚至可以一门课程都不申请。

因为不同的课程可能会被安排在不同的教室进行，所以牛牛需要利用课间时间从一间教室赶到另一间教室。

牛牛所在的大学有 \(v\) 个教室，有 \(e\) 条道路。每条道路连接两间教室，并且是可以双向通行的。由于道路的长度和拥堵程度不同，通过不同的道路耗费的体力可能会有所不同。 当第 \(i\)（\(1 \leq i \leq n-1\)）节课结束后，牛牛就会从这节课的教室出发，选择一条耗费体力最少的路径前往下一节课的教室。

现在牛牛想知道，申请哪几门课程可以使他因在教室间移动耗费的体力值的总和的期望值最小，请你帮他求出这个最小值。

\(\texttt{Data Range:}\)

保证 \(1 \le a_j, b_j \le v\)， \(1 \le w_j \le 100\)。

保证 \(1 \leq n \leq 2000\)，\(0 \leq m \leq 2000\)，\(1 \leq v \leq 300\)，\(0 \leq e \leq 90000\)。

Sol

每两个教室之间的最短路用 Floyd 处理，复杂度 \(O(n^3)\)。

整个耗费的体力值的期望可以拆成 \(i\) 到 \(i+1\) 的期望之和。要求出整个期望的最小值，因此要分开计算每种选择的期望。暴力枚举选择需要 \(\binom nm \times n\) 的时间。

发现在某节课做出的选择仅被已经申请调换的课程数限制，而与具体选择无关，因此考虑 DP。设 \(f(i,k)\) 表示前 \(i\) 节课申请换 \(k\) 门课时期望的最小值。发现期望与上一个时间点有无申请调课有关，因此加维记录是否调课。用 \(s\) 记录是否申请调课，\(s=0\) 表示未申请，\(s=1\) 表示申请。这样 DP 方程就很容易写出了。

虽然两个随机变量的期望和的有公式，且不要求两个随机变量相互独立。但“选择”与“不选择”不能用概率来衡量，用 DP 才能求出最优解。

【题解】P1081 [NOIP2012 提高组] 开车旅行

 2021-06-22 2021-06-24 动态规划 / 题解 / 倍增优化 DP / NOIP / 2012 10 分钟读完 (大约1448个字)

Des

小 \(\text{A}\) 和小 \(\text{B}\) 决定利用假期外出旅行，他们将想去的城市从 $1 $ 到 \(n\) 编号，且编号较小的城市在编号较大的城市的西边，已知各个城市的海拔高度互不相同，记城市 \(i\) 的海拔高度为\(h_i\)，城市 \(i\) 和城市 \(j\) 之间的距离 \(d_{i,j}\) 恰好是这两个城市海拔高度之差的绝对值，即 \(d_{i,j}=|h_i-h_j|\)。

旅行过程中，小 \(\text{A}\) 和小 \(\text{B}\) 轮流开车，第一天小 \(\text{A}\) 开车，之后每天轮换一次。他们计划选择一个城市 \(s\) 作为起点，一直向东行驶，并且最多行驶 \(x\) 公里就结束旅行。

小 \(\text{A}\) 和小 \(\text{B}\) 的驾驶风格不同，小 \(\text{B}\) 总是沿着前进方向选择一个最近的城市作为目的地，而小 \(\text{A}\) 总是沿着前进方向选择第二近的城市作为目的地（注意：本题中如果当前城市到两个城市的距离相同，则认为离海拔低的那个城市更近）。如果其中任何一人无法按照自己的原则选择目的城市，或者到达目的地会使行驶的总距离超出 \(x\) 公里，他们就会结束旅行。

在启程之前，小 \(\text{A}\) 想知道两个问题：

1、 对于一个给定的 \(x=x_0\)，从哪一个城市出发，小 \(\text{A}\) 开车行驶的路程总数与小 \(\text{B}\) 行驶的路程总数的比值最小（如果小 \(\text{B}\) 的行驶路程为 \(0\)，此时的比值可视为无穷大，且两个无穷大视为相等）。如果从多个城市出发，小 \(\text{A}\) 开车行驶的路程总数与小 \(\text{B}\) 行驶的路程总数的比值都最小，则输出海拔最高的那个城市。

2、对任意给定的 \(x=x_i\) 和出发城市 \(s_i\)，小 \(\text{A}\) 开车行驶的路程总数以及小 \(\text B\) 行驶的路程总数。

\(\texttt{Data Range:}\)

对于 \(30\%\) 的数据，有\(1\le n \le 20,1\le m\le 20\)；
对于\(40\%\) 的数据，有\(1\le n \le 100,1\le m\le 100\)；
对于 \(50\%\) 的数据，有\(1\le n \le 100,1\le m\le 1000\)；
对于 \(70\%\) 的数据，有\(1\le n \le 1000,1\le m\le 10^4\)；
对于 \(100\%\) 的数据：\(1\le n,m \le 10^5\)，\(-10^9 \le h_i≤10^9\)，\(1 \le s_i \le n\)，\(0 \le x_i \le 10^9\)
数据保证 \(h_i\) 互不相同。

Sol

首先预处理每个城市到东边最近的、次近的城市的位置。可以用线段树/平衡树/set/双向链表。如果用双向链表，不能用 STL 的，需要静态存储，因为需要知道每个城市在内存中的位置。综合考虑用 set 最方便。

设 \(f(i,s,k)\) 存储在第 i 个城市，轮到 \(s\) 开车（0 为 A，1 为 B），经过 \(2^k\) 天到达的城市以及走过的路程，那么 \(f(i,s,k)\) 可以倍增推导。

设 \(getDis(a,x)\) 获取从 \(a\) 城市出发，总路程最大为 \(x\) 时 A 和 B 驾驶的距离。那么 \(x\) 可以被二进制拆分，直到无论走几天需要经过的路程都小于 \(x\) 的剩余值。由于二进制拆分可以只用到 \(f(i,0,k)\)，所以只用对于 \(s=1\)，只用处理 \(k=0\) 的情况。

思路比较简单，就是有点难写。

Des

欧几里德的两个后代 Stan 和 Ollie 正在玩一种数字游戏，这个游戏是他们的祖先欧几里德发明的。给定两个正整数 \(M\) 和 \(N\)，从 Stan 开始，从其中较大的一个数，减去较小的数的正整数倍，当然，得到的数不能小于 \(0\)。然后是 Ollie，对刚才得到的数，和 \(M,N\) 中较小的那个数，再进行同样的操作……直到一个人得到了 \(0\)，他就取得了胜利。下面是他们用 \((25,7)\) 两个数游戏的过程：

Start:\((25,7)\)

Stan:\((11,7)\)

Ollie:\((4,7)\)

Stan:\((4,3)\)

Ollie:\((1,3)\)

Stan:\((1,0)\)

Stan 赢得了游戏的胜利。

整除状态即其中一个数整除另一个数的状态，先手必胜。

b<a<2b 的状态称为“决策单一状态“。比如求 25 和 7 的最大公因数的步骤

中的第 2 步和第 3 步是决策单一状态。

其他的状态称为决策不单一状态。

如果先手进入决策不单一状态，可以直接进入决策单一状态，也可以可以将 7 3 之流化为 4 3，迫使下一个人进入决策单一状态。那么先手可以随意操控后面行动的奇偶性，包括进入下一个决策不单一状态。所以只要决策不单一状态先手必胜。

那么只需要看初始状态到第一个非决策单一状态的距离，如果这个距离为奇数，那么先手必败，否则先手必胜。

【题解】P3306 [SDOI2013] 随机数生成器

 2021-06-01 2021-06-18 数论 / 同余 / BSGS / 题解 / 省选 / SDOI 6 分钟读完 (大约887个字)

Des

最近小 W 准备读一本新书，这本书一共有 \(p\) 页，页码范围为 \(0 \sim p-1\)。

小 W 很忙，所以每天只能读一页书。为了使事情有趣一些，他打算使用 NOI2012 上学习的线性同余法生成一个序列，来决定每天具体读哪一页。

我们用 \(x_i\) 来表示通过这种方法生成出来的第 \(i\) 个数，也即小 W 第 \(i\) 天会读哪一页。这个方法需要设置 \(3\) 个参数 \(a,b,x_1\)，满足 \(0\leq a,b,x_1\lt p\)，且 \(a,b,x_1\) 都是整数。按照下面的公式生成出来一系列的整数： \[x_{i+1} \equiv a \times x_i+b \pmod p\] 其中\(mod\)表示取余操作。

但是这种方法可能导致某两天读的页码一样。

小 W 要读这本书的第 \(t\) 页，所以他想知道最早在哪一天能读到第 \(t\) 页，或者指出他永远不会读到第 \(t\) 页。

\(\texttt{Data Range:}\)

对于全部的测试点，保证：

  • \(1 \leq T \leq 50\)。
  • \(0 \leq a, b, x_1, t \lt p\)，\(2 \leq p \leq 10^9\)。
  • \(p\) 为质数。

Sol

本来以为要用一堆矩乘魔幻操作，结果 llmmkk 实在太强了，1 秒钟就指出这种递推式可以转化为等比数列。我诧异地以为 llmmkk 已经完整掌握了高中数学必修五的知识，可他告诉我他是从 360 百科上学的，这就是神犇吗！

若 \(a_{n+1}=pa_n+q\)，设 \(a_{n+1}+\lambda=p(a_n+\lambda)\)，有 \(a_{n+1}=pa_n+(p-1)\lambda\)，则 \(\lambda=\dfrac q{p-1}\)。那么有 \(a_{n+1}+\dfrac q{p-1}=p(a_n+\dfrac q{p-1})\)，数列 \(\{a_{n+1}+\dfrac q{p-1}\}\) 是首项为 \({a_{1}+\dfrac q{p-1}}\)，公比为 \(p\) 的等比数列。

那么由题目可以得到

\[ (x_1+\dfrac b{a-1})\times a^{i-1}-\dfrac b {a-1}\equiv t\pmod p, \]

随便乱搞一下就可以用 BSGS 求解了。

丧心病狂的数据居然可能为 0。那么在 \(a=1,0\) 或 \(x_1+\dfrac b{a-1}=0\) 时都要特判，因为根本不能构成等比数列，甚至可能是等差数列。BSGS 内部也要特判 \(a\) 和 \(m\) 是否为 0。

Des

求同余方程 \[ \begin{cases} \begin{matrix} a_1-x\times K_1\equiv 0 \pmod {p_1}\\ a_2-x\times K_2\equiv 0 \pmod {p_2}\\ \cdots\\ a_n-x\times K_n\equiv 0 \pmod {p_n} \end{matrix} \end{cases} \] 的解，无解输出 \(-1\)。并且要满足 \[ \begin{cases} \begin{matrix} a_1-x\times K_1\le 0\\ a_2-x\times K_2\le 0\\ \cdots\\ a_n-x\times K_n\le 0 \end{matrix} \end{cases} \] 其中 \(K_i\) 可以在 \(O(\log n)\) 的时间获取。共 \(T\) 组数据。 \(\texttt{Data Range:}\)

对于所有的测试点，\(T \le 5\)，所有武器的攻击力 \(\le 10^6\)，所有 \(p_i\) 的最小公倍数 \(\le 10^{12}\)。

保证 $ T, n, m $ 均为正整数。

Sol

这题做了 4 天。

数据分为两种，一种满足 \(a_i\le p_i\)，一种满足 \(p_i=1\)。这题在解同余方程的基础上还要保证不等式小于 0，需要将执行 \(x\gets \max(x,\left\lceil\dfrac {K_i}{a_i}\right\rceil)\)。但是满足特性 1 的数据好像没有出现 \(p_i\) 全部等于 1 的情况，所以在这种情况没有执行特判也过了。

同余方程的形式是

\[ K_i\times x\equiv a_i \pmod {p_i},\tag1 \]

这也是本题最大的难点，许多题解用 exgcd 把解推出来。我这里提供一个将形如 \(ax\equiv p\pmod m\) 的同余方程转为 \(x\equiv a'\pmod m'\) 的方法。

首先把同余方程化为不定方程 \(ax+my=p\)，方程两边同除 \(\gcd(a,m,p)\)。设 \(\gcd(a,m,p)=d\)，得到 \[ \cfrac {a}d\cdot x+\cfrac {m}d\cdot y=\cfrac pd,\tag2 \]

此时的 \(\gcd(\cfrac ad,\cfrac md,\cfrac pd)=1\)，可得到 \(\gcd(\gcd(\cfrac ad,\cfrac md),\cfrac pd)=1\)，也就是说 \(\gcd(\cfrac ad,\cfrac md)\) 与 \(\cfrac pd\) 互质。

由裴蜀定理， \(\gcd(\cfrac ad,\cfrac md)\mid \cfrac pd\) 是式 (2) 有解的充要条件，即 \(\gcd(\cfrac ad,\cfrac md)=1\)，等价于 \(\cfrac ad\) 在 \(\bmod \cfrac md\) 意义下有逆元。那么，如果我们可以求出 \(\cfrac ad\) 在 \(\bmod \cfrac md\) 意义下的逆元，原方程就有解。否则，原方程无解。

那么式 (1) 就可以转化为 \[ x\equiv inv(\cfrac {K_i}d)\times \cfrac {a_i}d\pmod {\cfrac {p_i}d}, \] 将所有同余方程求出后，就可以用扩展中国剩余定理求解。

另外，std::set 和 std::map 使用 lower_bound 或 upper_bound 必须用成员函数的方法，否则复杂度不保证是 \(O(\log n)\)。

【题解】P1005 [NOIP2007 提高组] 矩阵取数游戏

 2021-05-19 2021-06-22 动态规划 / 基础算法 / 区间 DP / 题解 / 高精度 / NOIP / 2007 6 分钟读完 (大约925个字)

Des

帅帅经常跟同学玩一个矩阵取数游戏：对于一个给定的 \(n \times m\) 的矩阵，矩阵中的每个元素 \(a_{i,j}\) 均为非负整数。游戏规则如下：

  1. 每次取数时须从每行各取走一个元素，共 \(n\) 个。经过 \(m\) 次后取完矩阵内所有元素；
  2. 每次取走的各个元素只能是该元素所在行的行首或行尾；
  3. 每次取数都有一个得分值，为每行取数的得分之和，每行取数的得分 = 被取走的元素值 \(\times 2^i\)，其中 \(i\) 表示第 \(i\) 次取数（从 \(1\) 开始编号）；
  4. 游戏结束总得分为 \(m\) 次取数得分之和。

帅帅想请你帮忙写一个程序，对于任意矩阵，可以求出取数后的最大得分。

\(\texttt{Data Range:}\)

\(60\%\) 的数据满足：\(1\le n,m\le 30\)，答案不超过 \(10^{16}\)。

\(100\%\) 的数据满足：\(1\le n,m\le 80\)，\(0\le a_{i,j}\le1000\)。

Sol

脑抽了想来写个高精度。

看错题了，以为是随便取元素，不就是排序然后裸高精度嘛！

不过加上只能取行首或行尾这个条件后也不难，取过的元素可以“合并”到一起，变成一个得分和，然后就是区间 DP 那一套了。

不过发现我的做法和大多数题解不一样。单独考虑每行，设 \(f(i,j)\) 表示已经从左边取了 \(i\) 个数，从右边取了 \(j\) 个数时得分的最大值。转移方程为 \(f(i,j)=\max (f(i-1,j)+2^{i+j}\times a[i],f(i,j-1)+2^{i+j}\times a[m-j+1])\)。

普通的区间 DP 都是内部区间变化，但这个是双端变化，所以理论上这样会好做一些。

高精度写的很烂，其实应该按照正常 int 逻辑重载乘号和加号。
