A

\(O(n^2)\) DP 很简单。设 \(u\) 为当前节点，然后一直在思考如何用数据结构维护 \(\max (dp(y_i)+w[u] \operatorname{op}w[y_i])\)，\(y_i\) 是 \(u\) 的祖先节点。

但是，这玩意是二进制按位运算，不是没有规律的东西啊。。。

对于 xor 和 or，有 \(a \operatorname{op} b\le a\operatorname{op} c + c\operatorname{op} b\)（or 显然。xor 可以按位考虑，当 \(a,b\) 的同一位不同，\(c\) 的那一位必和 \(a,b\) 其中一个不同。当 \(a,b\) 某一位相同，\(c\) 可能相同，可能不同）。

对于 and，也有 \(a \operatorname{op} b\le a\operatorname{op} c + c\operatorname{op} b\)，但这里要求 \(a \and b\) 的位数和 \(a\and c\) 的位数一致。

那么，xor 和 or 的情况直接取从根节点下来的链。and 需要记录某一位有 1、且离当前节点最近的那个节点。

时间复杂度 \(O(n\log w)\)
