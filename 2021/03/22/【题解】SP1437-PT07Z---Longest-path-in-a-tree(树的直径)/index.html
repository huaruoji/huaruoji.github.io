Des

题目描述 给你一个无权无向的树。编写程序以输出该树中最长路径（从一个节点到另一个节点）的长度。在这种情况下，路径的长度是我们从开始到目的地的遍历边数。

输入输出格式 输入格式： 输入文件的第一行包含一个整数N ---树中的节点数（0 < N <= 10000）。接下来N -1行包含该树的N -1个边---每行包含一对（u，v），表示在节点u和节点v之间存在边（1 <= u，v <= N）。

输出格式： 一行：输出最长路径的长度。
