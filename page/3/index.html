O(n^2)的基础算法,如果是不下降只用把f[i]<f[j]改成f[i]<=f[j]

2. 最长公共子序列(LCS)

给定一个序列X=<x1,x2,x3,x4...,xm>，另一个序列Z=<z1,z2,z3,z4...,zk>，若存在一个严格递增的X的下标序列<i1,i2,i3,…,ik>对所有的1,2,3,…,k，都满足x(ik)=zk，则称Z是X的子序列

比如Z=<B,C,D,B>是X=<A,B,C,B,D,A,B>的子序列

#include <iostream>
#include <cstring>

#include <algorithm>

using namespace std;

int main()
{
ios::sync_with_stdio(false);

if\(n==1\) cout\<\<ans;
else
\{
    for\(int j=2;j\<=n;j++\)
    \{
        if\(j\%2==0\)
        \{
            ans=ans\*2+1;
        \}
        else
        \{

        \}

    \}
\}

}

{

最小生成树

最小生成树就是在一个图中寻找一个联通图,必须包含原图的所有节点,且这个图的所有边的权值和最小.
为什么是一个树呢?因为只要求联通,就一定没有环,没有环不就是树了嘛!

Kruskal算法

Kruskal算法的核心是加边和判环.一条一条把边加上,如果加上会形成环就不加,最后弄成一个连通图.

假设有n个节点,就需要加(n-1)条边.为什么呢?因为每加上一条边,就会多连接一个点.
因为是要求最小生成树,所以先将所有边权值从小到大排序,挨个加边,加完的值一定是最小的.
判环主要用并查集实现,若两个节点的祖先不同,则加边后不会形成环,反之会形成环.
