【题解】P1288 取数游戏II

 2021-06-05 2021-06-05 数学 / 博弈论 5 分钟读完 (大约710个字)

Des

有一个取数的游戏。初始时，给出一个环，环上的每条边上都有一个非负整数。这些整数中至少有一个0。然后，将一枚硬币放在环上的一个节点上。两个玩家就是以这个放硬币的节点为起点开始这个游戏，两人轮流取数，取数的规则如下：

（1）选择硬币左边或者右边的一条边，并且边上的数非0；

（2）将这条边上的数减至任意一个非负整数(至少要有所减小)；

（3）将硬币移至边的另一端。

如果轮到一个玩家走，这时硬币左右两边的边上的数值都是0，那么这个玩家就输了。

如下图，描述的是Alice和Bob两人的对弈过程，其中黑色节点表示硬币所在节点。结果图(d)中，轮到Bob走时，硬币两边的边上都是0，所以Alcie获胜。

（a）Alice （b）Bob （c）Alice （d）Bob

现在，你的任务就是根据给出的环、边上的数值以及起点（硬币所在位置），判断先走方是否有必胜的策略。

\(\texttt{Data Range:}\)

第一行一个整数N（N≤20），表示环上的节点数。

第二行N个数，数值不超过30，依次表示N条边上的数值。硬币的起始位置在第一条边与最后一条边之间的节点上。

Sol

分情况讨论。

一、当前状态有一边是 0

那么必然会将另一边全部取为 0，否则必败。在取为 0 的基础上，如果到下一个 0 边的距离为奇数，那么必胜，否则必败。

二、当前状态两边都是 0

必败。

三、当前状态两边都不是 0

如果向右走到第一个 0 边的距离为奇数，那么把右边的边取为 0 必胜。向左走也是同样。

如果距离下一个 0 边的距离为偶数，那么不管怎么取都必败（会让下一个人进入状态二或状态三）。

总结一下，状态一、二、三其实可以合并，只要计算向左或向右到第一个 0 边的距离。如果有一个为奇数必胜，否则必败。

【笔记】康托展开

 2021-06-02 2021-06-03 组合数学 / 康托展开 1 分钟读完 (大约152个字)

康托展开

康托展开用于求出一个 \(1\sim N\) 的排列在所有排列中的排名。

枚举每一位，并给排名加上当前位更小的数的总数。由于前面位置上的数可能比当前位置上的数小，需要使用树状数组记录。

逆康托展开

由于 \(n!>(n-1)\times (n-1)!+(n-2)\times(n-2)!\dots+1\)，使用排名整除上对应的阶乘就可以得到某位是多少。

证明 \(n!-(n-1)\times (n-1)!=(n-1)!\)，\((n-1)!-(n-2)\times(n-2)!=(n-2)!\)，最后会得到 1。也可以用归纳法。

【题解】P3306 [SDOI2013] 随机数生成器

 2021-06-01 2021-06-18 数论 / 同余 / BSGS / 题解 / 省选 / SDOI 6 分钟读完 (大约887个字)

Des

最近小 W 准备读一本新书，这本书一共有 \(p\) 页，页码范围为 \(0 \sim p-1\)。

小 W 很忙，所以每天只能读一页书。为了使事情有趣一些，他打算使用 NOI2012 上学习的线性同余法生成一个序列，来决定每天具体读哪一页。

我们用 \(x_i\) 来表示通过这种方法生成出来的第 \(i\) 个数，也即小 W 第 \(i\) 天会读哪一页。这个方法需要设置 \(3\) 个参数 \(a,b,x_1\)，满足 \(0\leq a,b,x_1\lt p\)，且 \(a,b,x_1\) 都是整数。按照下面的公式生成出来一系列的整数： \[x_{i+1} \equiv a \times x_i+b \pmod p\] 其中\(mod\)表示取余操作。

但是这种方法可能导致某两天读的页码一样。

小 W 要读这本书的第 \(t\) 页，所以他想知道最早在哪一天能读到第 \(t\) 页，或者指出他永远不会读到第 \(t\) 页。

\(\texttt{Data Range:}\)

对于全部的测试点，保证：

  • \(1 \leq T \leq 50\)。
  • \(0 \leq a, b, x_1, t \lt p\)，\(2 \leq p \leq 10^9\)。
  • \(p\) 为质数。

Sol

本来以为要用一堆矩乘魔幻操作，结果 llmmkk 实在太强了，1 秒钟就指出这种递推式可以转化为等比数列。我诧异地以为 llmmkk 已经完整掌握了高中数学必修五的知识，可他告诉我他是从 360 百科上学的，这就是神犇吗！

若 \(a_{n+1}=pa_n+q\)，设 \(a_{n+1}+\lambda=p(a_n+\lambda)\)，有 \(a_{n+1}=pa_n+(p-1)\lambda\)，则 \(\lambda=\dfrac q{p-1}\)。那么有 \(a_{n+1}+\dfrac q{p-1}=p(a_n+\dfrac q{p-1})\)，数列 \(\{a_{n+1}+\dfrac q{p-1}\}\) 是首项为 \({a_{1}+\dfrac q{p-1}}\)，公比为 \(p\) 的等比数列。

那么由题目可以得到

\[ (x_1+\dfrac b{a-1})\times a^{i-1}-\dfrac b {a-1}\equiv t\pmod p, \]

随便乱搞一下就可以用 BSGS 求解了。

丧心病狂的数据居然可能为 0。那么在 \(a=1,0\) 或 \(x_1+\dfrac b{a-1}=0\) 时都要特判，因为根本不能构成等比数列，甚至可能是等差数列。BSGS 内部也要特判 \(a\) 和 \(m\) 是否为 0。

【笔记】BSGS 算法

 2021-06-01 2021-06-18 数论 / 同余 / BSGS 6 分钟读完 (大约969个字)

BSGS 算法（Baby-Step, Giant-Step）

求解 \(a^x\equiv m\pmod p\)，即离散对数的算法。使用普通 BSGS 算法的前提条件是 \(a,p\) 互质。

由于 \(a\) 模 \(p\) 的次数小于等于 \(\varphi(p)\)，所以朴素算法的复杂度是 \(O(\varphi(p))\)。

设 \(a^1,a^2,a^3,\dots,a^k\) 已知，\(1\le t\le k\)。由于 \(a^{k+t}\equiv m\pmod p\) 可转化为 \(a^t\equiv m\times a^{-k}\pmod p\)，若 \(a^t\) 存在， \(x=k+t\) 就是原方程的解。枚举 \(f=0,k,2k,\dots\)，再检验是否有 \(a^{f+t}\equiv m\pmod p\) 存在，便可遍历完 \(a^x\) 的循环节。这样做的复杂度是 \(O(k+\dfrac pk)\)，当 \(k=\sqrt p\) 时最优。
