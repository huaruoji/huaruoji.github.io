费马小定理

定理 \(a\) 不是质数 \(p\) 的倍数，有 \[ a^{p-1}\equiv 1\pmod p. \] 但是费马小定理的逆定理并不成立，也就是说，竟然存在一个合数 \(p\)，使得所有的 \(a^{p-1}\) 模 \(p\) 同余 1。这样的合数称作卡迈克尔数，最小的卡迈克尔数是 \(561=3\times 11\times 17\)。

二次探测定理

定理 若 \(p\) 是一个奇素数，且 \(0<x<p\)，则方程 \(x^2=1\pmod p\) 的解是 \(x_1=1,x_2=p-1\)。

证明 将方程移项并因式分解，得到 \((x-1)(x+1)=0\pmod p\)。有 \(p\mid (x-1)(x+1)\)，由于 \(p\) 是奇素数，得到 \(x=1\) 或 \(x=p-1\)。

【题解】P1242 新汉诺塔

 2021-05-27 2021-05-27 基础算法 / 搜索 / 策略 2 分钟读完 (大约340个字)

Des

设有 \(n\) 个大小不等的中空圆盘，按从小到大的顺序从 \(1\) 到 \(n\) 编号。将这 \(n\) 个圆盘任意的迭套在三根立柱上，立柱的编号分别为 \(A\) , \(B\) , \(C\)，这个状态称为初始状态。

现在要求找到一种步数最少的移动方案，使得从初始状态转变为目标状态。

移动时有如下要求：

  • 一次只能移一个盘；
  • 不允许把大盘移到小盘上面。

\(\texttt{Data Range:}\)

对于 \(100\%\) 的数据，\(1 \le n \le 45\) ，$1 $ 每个圆盘的编号 \(\le n\) 。

每行的圆盘描述是从下到上的圆盘编号。

Sol

u[x] 记录盘子现在在的柱子，v[x] 记录要去到的柱子。

外层循环从较大的盘子开始枚举，调用 get(x, v[x]) 使得 x 到达目标盘。这样一定比先调用小盘子优。

get 内部需要将小于 x 的盘子都移动到一个空闲位置，但这里有 2 种策略，不能保证某个一定更优。

  1. 将 x 移动到目标柱，小盘子移动到 6 - v[x] - u[x] 柱子。
  2. 将 x 移动到目标柱，小盘子保留在原柱子。

分 2 次 DFS，取最优值。

B CF1500E Subset Trick

首先，如果要判断能否挑出一个和 \(\le x\) 的集合，必然考虑排序后从小到大选数。否则可能挑不出来。也就是说，设排序后的前缀和数组为 \(L\)，后缀和数组为 \(R\)（\(L_i\) 表示前 \(i\) 个数的和，\(R_i\) 表示后 \(i\) 个数的和），可以取到的 \(x\) 为 \([L_i,R_i)\) 的并。

每次操作会更新 \(n\) 级别的区间，朴素求一次区间并需要 \(O(n\log n)\) 的时间，考虑优化。

如果任意 2 个相邻的区间的交不为空，那么答案为 \(R_{n-1} - L_1 -1\)，这也是特殊性质 \(A,B\) 满足的条件。反之，答案为 \(\sum_{i=1}^n R_i-L_i-1\)。

所有的区间不一定满足上述两个性质，那退而求其次，一部分的区间满不满足（毕竟满足上面 2 个性质中任意一个，这些区间的贡献就可以 \(O(\log n)\) 求出）呢？

2 个相邻的区间相交且实际上与 \(L_{i+1}<R_i\) 等同，也就是 \(L_{i+1}-R_i<0\)。\(L_i\) 和 \(R_i\) 都是单调递增的，但 \(L_i\) 增长的更快。所以，\(g(x)=L_{i+1}-R_i\) 一定是一个先减后增的单峰函数（画出图像就知道了），但零点个数不能确定。

三分（用差分转换为二分）找到函数极值点后，在极值点左右分别二分找到零点，就可以分 3 段统计答案。时间复杂度 \(O(n\log ^2n)\)。

详细说说如何计算。如果只有一个零点，那么必然为存储的第一个，零点左边的使用第二种方式计算。如果有第二个零点，2 个零点之间的用第一种方式计算，第二个零点右边的用第二种方式计算。如果一个零点也没有，全部按第一种方式计算。

另外，题解说中间变量可能会爆 long long。虽然我觉得不会，但也是学到了如果最终答案在 long long 范围内，不用管中间变量会不会爆，可以得到正确答案（其实就是模运算的性质）。

【题解】P4774 [NOI2018] 屠龙勇士

 2021-05-24 2021-06-10 数论 / 同余 / 线性同余方程 / 题解 / NOI / 2018 8 分钟读完 (大约1131个字)

Des

求同余方程 \[ \begin{cases} \begin{matrix} a_1-x\times K_1\equiv 0 \pmod {p_1}\\ a_2-x\times K_2\equiv 0 \pmod {p_2}\\ \cdots\\ a_n-x\times K_n\equiv 0 \pmod {p_n} \end{matrix} \end{cases} \] 的解，无解输出 \(-1\)。并且要满足 \[ \begin{cases} \begin{matrix} a_1-x\times K_1\le 0\\ a_2-x\times K_2\le 0\\ \cdots\\ a_n-x\times K_n\le 0 \end{matrix} \end{cases} \] 其中 \(K_i\) 可以在 \(O(\log n)\) 的时间获取。共 \(T\) 组数据。 \(\texttt{Data Range:}\)

对于所有的测试点，\(T \le 5\)，所有武器的攻击力 \(\le 10^6\)，所有 \(p_i\) 的最小公倍数 \(\le 10^{12}\)。

保证 $ T, n, m $ 均为正整数。

Sol

这题做了 4 天。

数据分为两种，一种满足 \(a_i\le p_i\)，一种满足 \(p_i=1\)。这题在解同余方程的基础上还要保证不等式小于 0，需要将执行 \(x\gets \max(x,\left\lceil\dfrac {K_i}{a_i}\right\rceil)\)。但是满足特性 1 的数据好像没有出现 \(p_i\) 全部等于 1 的情况，所以在这种情况没有执行特判也过了。

同余方程的形式是

\[ K_i\times x\equiv a_i \pmod {p_i},\tag1 \]

这也是本题最大的难点，许多题解用 exgcd 把解推出来。我这里提供一个将形如 \(ax\equiv p\pmod m\) 的同余方程转为 \(x\equiv a'\pmod m'\) 的方法。

首先把同余方程化为不定方程 \(ax+my=p\)，方程两边同除 \(\gcd(a,m,p)\)。设 \(\gcd(a,m,p)=d\)，得到 \[ \cfrac {a}d\cdot x+\cfrac {m}d\cdot y=\cfrac pd,\tag2 \]

此时的 \(\gcd(\cfrac ad,\cfrac md,\cfrac pd)=1\)，可得到 \(\gcd(\gcd(\cfrac ad,\cfrac md),\cfrac pd)=1\)，也就是说 \(\gcd(\cfrac ad,\cfrac md)\) 与 \(\cfrac pd\) 互质。

由裴蜀定理， \(\gcd(\cfrac ad,\cfrac md)\mid \cfrac pd\) 是式 (2) 有解的充要条件，即 \(\gcd(\cfrac ad,\cfrac md)=1\)，等价于 \(\cfrac ad\) 在 \(\bmod \cfrac md\) 意义下有逆元。那么，如果我们可以求出 \(\cfrac ad\) 在 \(\bmod \cfrac md\) 意义下的逆元，原方程就有解。否则，原方程无解。

那么式 (1) 就可以转化为 \[ x\equiv inv(\cfrac {K_i}d)\times \cfrac {a_i}d\pmod {\cfrac {p_i}d}, \] 将所有同余方程求出后，就可以用扩展中国剩余定理求解。

另外，std::set 和 std::map 使用 lower_bound 或 upper_bound 必须用成员函数的方法，否则复杂度不保证是 \(O(\log n)\)。

【笔记】中国剩余定理

 2021-05-21 2021-07-05 数论 / 同余 / 线性同余方程 10 分钟读完 (大约1508个字)

前置知识

如果 \(a\equiv b\pmod m,c\equiv d\pmod m\)，则有

① \(ax+cy\equiv bx+dy\pmod m\)，其中 \(x,y\) 为任给的整数。

② \(ac\equiv bd\pmod m\)。

③ \(a^n\equiv b^n\pmod m\)。

④ \(f(a)\equiv f(b)\pmod m\)，其中 \(f(x)\) 为任给的整系数多项式。

证明 ① 由 \(m\mid ax-bx,m\mid cy-dy\)，得到 \(m\mid ax+cy-bx-dy\)。

② 由 \(m\mid b(a-c)+c(b-d)=ab-cd\) 得到。

③ 由②可得到。

④ 由③和①可得到。

【笔记】关于 mod 运算

 2021-05-21 2021-07-07 数论 / 整除 1 分钟读完 (大约163个字)

数论中的 mod

首先定义带余除法为 \[ a=qb+r,0\le r < |b|. \] 上式中的 \(a\) 为被除数，\(b\) 为除数，\(r\) 为 \(a\) 除以 \(b\) 得到的余数，记作 \(a\bmod b=r\)。

部分数学领域定义的 mod

\[ a\bmod b=a-\lfloor a/b\rfloor \times b. \]

C++ 中的 mod

定义带余除法为 \[ a=qb+r,0\le |r| < |b|. \] 若 \(a\ge 0\)，则 \(r\ge 0\)。若 \(a<0\)，则 \(r<0\)。

区别

第一个定义和第二个定义是等价的，C++ 中的 mod 在 \(a\ge0\) 时和数学定义是一样的。因此扩展欧几里得算法只能处理 \(a,b\ge0\) 的情况。

【题解】P5431 【模板】乘法逆元2

 2021-05-20 2021-05-20 数学 / 其他 / 逆元 4 分钟读完 (大约590个字)

Des

给定 \(n\) 个正整数 \(a_i\) 与正整数 \(k\)，求模 \(p\) 意义下的 \[ \sum\limits_{i=1}^n\frac{k^i}{a_i}. \]

\(\texttt{Data Range:}\)

对于 \(30\) 分的数据：\(1\le n \le 10^5\) 对于全部数据：\(1\le n \le 5\times 10^6\) ，\(2\le k < p \le 10^9\) ，\(1\le a_i < p\) ，保证\(p\)为质数。

Sol

\(O(n\log n)\) 是没办法过这道题的，大衍求一术也不行（话说我怎么这么喜欢大衍求一术）。展开要求的式子： \[ \sum_{i=1}^n \frac {k^i}{a_i}=\frac {k^1}{a_i}+\frac {k^2}{a_2}+\dots +\frac {k^n}{a_n}, \] 很容易想到通分后就只用求一次逆元了，但这样必须要求每个 \(\frac {\prod_{i=1}^n a_i}{a_k}(1\le k\le n)\) 的值。我的思路就卡在这里了，认为这是无法 \(O(1)\) 求出来的。实际上预处理前缀积、后缀积就可以求出来。这方法也可以用于求任意 \(\prod_{i=1}^ka_i\times \prod_{i=q}^na_i(q>k)\) 的值。

另一种做法：

设 \(s_i=\prod_{k=1}^i a_k\)，则 \(\dfrac 1{a_i}=s_{i-1}\times \dfrac 1{s_{i}}\)。前缀积与前缀积的逆元都是可以用逆元模版 1 里的经典方法线性求出来的。
